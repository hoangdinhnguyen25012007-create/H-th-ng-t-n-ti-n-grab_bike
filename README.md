# 🛵 SmartÔm AI — Hệ thống tính tiền xe ôm

Ứng dụng tính giá xe ôm thông minh dựa trên **Fuzzy Logic (Hệ suy luận mờ)**,
tích hợp bản đồ thực tế và hệ thống điểm thưởng.

## ✨ Tính năng
- 🧠 Tính giá tự động bằng Fuzzy Inference System (Mamdani)
- 🗺️ Tính quãng đường thực tế qua OSMnx + NetworkX
- 📍 Hiển thị tuyến đường trên bản đồ với Folium
- ⭐ Hệ thống điểm thưởng thông minh cho tài xế
- 📊 Biểu đồ membership functions trực quan

## 🛠️ Công nghệ sử dụng
- Python 3.x
- scikit-fuzzy
- osmnx · networkx · folium · geopy
- tkinter (giao diện desktop)
